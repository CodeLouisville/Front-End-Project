## Student Challenge - Answers ##

### 1: 2 Column layout with Flexbox ###


  ```css

  .row {
  	display: flex;
  	flex-direction: row;
  	flex-wrap: wrap;
  	width: 100%;
  }

  .column {
  	display: flex;
  	flex-direction: column;
  	flex-basis: 100%;
  	flex: 1;
  }

  ```

### 2. Button ###

```css

.btn {
  background-color: #32A952;
	-moz-border-radius: 12px;
	-webkit-border-radius: 12px;
	border-radius: 12px;
	border: 1px solid #2FA74D;
	cursor: pointer;
	color: #FFFFFF;
	font-family: sans-serif;
	font-size: 1.25em;
	font-weight: bold;
	padding: 0.5em 1em;
	text-decoration: none;
	text-transform: uppercase;
	width: 100%;
	max-width: 300px;
	text-align: center;
	display: block;
}

.btn:hover {
	background-color: #288641;
}

```
